overzicht

dit project is een game die ik op stage heb gemaakt voor stichting Dorpsplatform Jong Elsloo.

de opdracht

Stichting Dorpsplatform Jong Elsloo wilde voor een activiteit een game gemaakt hebben die de mensen die naar de activiteit komen kunnen spelen. de game moest leuk blijven zelfs na een paar keer gespeeld te hebben en een leuke game loop hebben. er moest ook een verhaal lijn inzitten die de speler kon ontdekken door de game te spelen. verder moesten er ook bosses zijn, moeilijkere enemies die de speler moet verslaan. verder dingen die in de game moesten zitten om het leuker te maken zijn items voor de speler, leuke muziek en sound effects, visueel aantrekkelijke UI, en geen game breaking bugs.
